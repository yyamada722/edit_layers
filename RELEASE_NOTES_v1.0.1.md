# Edit Layers v1.0.1

初回パブリックリリースです。モデリングの手作業をレイヤーとして積み上げられる**非破壊メッシュ編集スタック**を Blender に追加します。

## ハイライト

- **なんでもレイヤーに記録** — 「記録」を押して普段どおり編集し「コミット」するだけ。押し出し・ベベル・サブディバイド・削除などトポロジ変更を含む編集セッションが 1 レイヤーとして保存されます。専用ツールは不要で、編集モード/スカルプトモードの標準機能がそのまま使えます
- **非破壊スタック** — レイヤーの表示切替・並べ替え・削除・途中レイヤーの再編集が自由。上流を変更すると下流が追従し、押し出し等の新規ジオメトリも近傍アンカー相対で付いてきます
- **ブランチ** — 任意のレイヤーから編集パターンを分岐。ワンクリックで切り替え、「並べて比較」でバリエーションを横に並べて検討できます。分岐点より上流の共有レイヤーを直すと全ブランチに波及します
- **影響ハイライト** — 選択レイヤーが動かした頂点 (橙) / 作った頂点 (緑) をビューポートに表示
- **レイヤーの整理** — 「直前のレイヤーと統合」「ここまでをベースに確定」(部分ベイク) でスタックを畳めます
- **記録し忘れの救済** — 記録せずに編集しても自動検出され、「レイヤーとして取り込む」で遡ってレイヤー化できます
- **属性も記録** — マテリアル割り当て・スムーズ/シャープ・シーム・クリース・ベベルウェイト
- **安全設計** — シェイプキーとの排他ロック、共有レイヤーの保護、壊れた参照は警告してスキップ (クラッシュしません)
- **日本語/英語 UI** — Blender の言語設定に自動追従

## 動作環境

- Blender 5.0 以降 (5.1.1 で動作確認)

## インストール

`edit_layers-1.0.1.zip` をダウンロードし、プリファレンス > 拡張機能 > 右上メニュー > **Install from Disk** でインストールしてください。パネルは 3D ビューポートのサイドバー (N キー) > Edit Layers タブに表示されます。ヘルプはパネル右下の **?** ボタンから開けます。

## 既知の制限

- UV と頂点カラーは記録されません
- 新規ジオメトリのアンカー追従は平行移動のみ (上流の回転・スケールには追従しません)
- シェイプキーとは併用できません (キー追加は自動でブロックされます。モデリング完了・ベイク後に作成してください)
- 記録中に Blender を終了すると、その記録セッションは失われます (コミット済みレイヤーは .blend に保存されます)
